# File: vault.txt
#
# This file is used to create the "vault template" information for Sil.
#
# Do not modify this file unless you know exactly what you are doing,
# unless you wish to risk possible system crashes and broken savefiles.
#
#
# === Understanding vault.txt ===
#
# name: vault name
# type: room type
# depth: depth
# rarity: rarity
# flags: flag1 | flag2 | flag3 ...
# D: lines giving full layout of vault using symbols below
#
# 'name' indicates the beginning of an entry.
#
# 'type' is for room type
#
#     type 10 - Gates of Angband
#     type 6  - interesting room
#     type 7  - lesser vault
#     type 8  - greater vault
#     type 9  - Morgoth's vault
#
# 'depth' is for mimnimum depth.
#
# 'rarity is for the vault's rarity.
#
# 'flags' is for the vault flags (if any)
#
#     TEST        - must be generated when choosing a vault (for debugging)
#     NO_ROTATION - must not be rotated when placing in the dungeon
#     TRAPS       - has twice the normal proportion of traps
#     WEBS        - has a 5% chance of a web per empty square
#     LIGHT       - 100% chance of being permanently lit
#
# 'D' lines describe the layout of the vault. Lines must be padded
#     with spaces to fill the dimensions specified. Vaults are described
#     with the following symbols:
#
#     $ - outside of the vault, where corridors may be connected
#     # - granite
#     % - quartz vein
#     : - rubble
#     ; - glyph of warding
#     > - down stairs
#     < - up stairs
#     + - visible door
#     s - secret door
#     ^ - trap (50% chance -- use sparingly as there is already a TRAPS flag)
#     0 - forge
#     7 - chasm
#     1 - monster +1 level
#     2 - monster +2 levels
#     3 - monster +3 levels
#     4 - monster +4 levels
#     * - treasure +1 to +4 levels
#     & - treasure +1 to +4 levels (good)
#     ~ - a chest with +4 levels
#     S - an elf skeleton
#     ? - a monster +1 level and/or an item +1 level
#     @ - easterling spy
#     H - silent watcher
#     o - orc champion
#     O - orc captain
#     f - cat warrior
#     F - cat assassin
#     t - cave troll
#     T - troll guard
#     W - barrow wight
#     a - archer +1 level (any monster that shoots arrows)
#     b - flier +1 level (any monster that flies)
#     c - wolf +1d4 levels
#     v - vampire +1d4 levels
#     r - raukar +1d4 levels
#     M - spider +1d4 levels
#     d - dragon +4 levels
#     A - Aldor
#     R - Gothmog
#     U - Ungoliant
#     D - Glaurung
#     G - Gorthaur
#     V - Morgoth
#     C - Carcharoth
#

# Future Vault Thoughts
#
# Ways to add more variety to the vault format:
#
#   Using a symbol such that one of that symbol becomes a secret door
#   e.g.
#
#   N:81:Ringway 2
#   X:6:5:8:10:14
#   D:$$$$$$$$$$$$
#   D:$..........$
#   D:$......?...$
#   D:$..#ssss#..$
#   D:$.?s.*.2s..$
#   D:$..s....s..$
#   D:$..#ssss#..$
#   D:$..........$
#   D:$...?......$
#   D:$$$$$$$$$$$$
#
#   maybe have lines:
#   O:y:33
#   M:u:42
#   T:):23
#
#   (or O:y:Mail Corslet)
#
#   for object, monster, terrain, with a symbol and then this is overridden in
#   the map so that it represents the index listed. One issue is things at a
#   coarse grained level currently: chest, archer, flier, wolf, vampire,
#   raukar, spider, dragon
#
# Maybe add a max-depth for vaults
#
